About this page:

Hello and welcome to this page. Here I'm trying to create a page from scratch for my photography. I am currently learning HTML and wanted to try implementing what I have learned with a hobby that I am passionate about. I'm an amateur photographer who likes to go on hikes and take landscape photography, macro shots, and anything in black and white. Currently I am using a Nikon D7100 body with 18-55mm and 55-300mm lenses.Hope you enjoy this site and subscribe at the bottom of this page!

Locations and Style:

  • Northeastern states

  • South India

The things I look for when taking photos are:

  1. Brightness/Lighting
  2. Color and contrast
  3. Stark features
  4. Hidden details that would
    otherwise go unnoticed

Trey Ratcliff's Photoblog

In addition to the actual content of this photo blog, I found the source code to be pretty interesting. This site's framework is written in HTML5. There were definitely some elements that I was not really familiar with when initially glancing over the source code. Some examples are the meta and script tags. After reading more about them, I learned that:

  • Meta tags and elements are mainly used to provide meta data about that existing HTML file. Some examples are keywords, authors, dates of modifications, etc... This type of data is used by search engines and web browsers.
  • Script tags and elements are used to refer to "client-side script" or "external scripts".
  • Source: w3schools.com

One possible area of improvement of the source code for this site is that it could be organized better. There were not very many indentations to help visually understand how and why the script was written the way it was. Source: w3schools.com
